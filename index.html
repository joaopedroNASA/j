Projeto 1

Este projeto foi desenvolvido usando HTML, CSS e JavaScript. O objetivo foi criar um site responsivo que se adapta a diferentes tamanhos de tela. Utilizei Flexbox e Grid para o layout, além de implementar interatividade com JavaScript, como efeitos de hover e validação de formulários.

Projeto 2

No Projeto 2, criei uma aplicação web dinâmica utilizando React e Node.js. A ideia foi construir uma plataforma de gerenciamento de tarefas, onde os usuários podem adicionar, editar e excluir tarefas. O back-end foi desenvolvido com Node.js e Express, enquanto o front-end foi feito com React.
